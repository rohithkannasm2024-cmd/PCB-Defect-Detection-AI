# 🤖 AI PCB Defect Detection using YOLOv8

An AI-powered web application for detecting PCB manufacturing defects using YOLOv8 and Flask.

## Features

- Upload PCB images
- Detect PCB defects using AI
- Display detected defects with bounding boxes
- Confidence score
- Responsive web interface

## Technologies Used

- Python
- Flask
- Ultralytics YOLOv8
- OpenCV
- HTML
- CSS

## Installation

```bash
git clone <repository-url>

cd PCB-Defect-Detection-AI

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python app.py
```
